cpbitmap.github.io

cpbitmap.github.io is a free browser based tool that can convert cpbitmap files to multiple image formats.

What is a cpbitmap file?

The cpbitmap file format is a proprietary file format created by Apple to store iOS background images.

How do we know how to convert cpbitmap files?

Apple has never released any documentation about the layout of cpbitmap files. We only know how to convert cpbitmap images due to some clever guessing and trial and error by Simon Whitehead in 2011.

There are a few scripts that use his description to convert cpbitmap images to other formats, but I couldn't find a working online version. The code used in this tool was originally written by dr15 in this Stack Overflow answer to work with Node.js, but has been slightly edited to run in the browser.
